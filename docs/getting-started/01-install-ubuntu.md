# 🐧 Install Ubuntu Server

## Goal

Install a fresh Ubuntu Server 24.04 LTS.

---

## Requirements

- USB Flash Drive
- Ubuntu Server ISO

---

## Steps

1. Download Ubuntu Server.
2. Create a bootable USB.
3. Install Ubuntu.
4. Create your first user.
5. Enable OpenSSH during installation.
6. Update the system.

---

## Verification

```bash
lsb_release -a
```

Expected:

```
Ubuntu 24.04 LTS
```
