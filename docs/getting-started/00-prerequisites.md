# 📋 Prerequisites

Before installing HomeServer, make sure your system meets the following requirements.

---

## Hardware

| Component | Minimum | Recommended |
|-----------|---------|-------------|
| CPU | Dual Core | Intel Core i3+ |
| RAM | 4 GB | 8 GB |
| Storage | 40 GB | 120 GB SSD |
| Network | Internet | Broadband |

---

## Operating System

- Ubuntu Server 24.04 LTS

---

## Required Software

- Git
- Docker Engine
- Docker Compose Plugin

---

## Accounts

- GitHub Account
- Cloudflare Account (Optional)

---

## Domain

Optional

Example:

- example.com

---

## Result

After completing this guide, your machine will be ready to install HomeServer.
